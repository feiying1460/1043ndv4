Internet Help

Internet Connection Type: L2TP

If your ISP is running on DHCP server, please select Dynamic IP.

If your ISP provides a Static IP Address, Subnet Mask, Gateway and DNS Settings, please select Static IP.

If your ISP provides a PPPoE connection, please select PPPoE.

If your ISP provides a BigPond Cable (or Heart Beat Signal) connection, please select BigPond Cable.

If your ISP provides a L2TP connection, please select L2TP.

If your ISP provides a PPTP connection, please select PPTP.

VPN Server IP/Domain Name - Enter the Server IP Address or Domain Name provided by your ISP.

User Name/Password - Enter the User Name and Password provided by your ISP. These fields are case-sensitive.

Dynamic IP/Static IP - Select Static IP if the IP Address, Subnet Mask, Gateway and DNS Server Address are provided by your ISP. Otherwise, please select Dynamic IP.

IP address - Enter the IP Address provided by your ISP (only if Static IP is selected).

Subnet Mask - Enter the Subnet Mask provided by your ISP (only if Static IP is selected).

Default Gateway - Enter the Default Gateway provided by your ISP (only if Static IP is selected).

DNS - Enter the DNS Server IP Address provided by your ISP (only if Static IP is selected).

Click Connect to connect immediately.

Click Disconnect to disconnect immediately.

Click Save to save all your settings.
